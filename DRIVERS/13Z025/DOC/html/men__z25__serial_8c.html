Register 16Z025/125 UARTs at serial8250 core driver.

Author:
kp/ub/ts

Switches: CONFIG_MEN_Z025_UART_BASECLK - Set default base clock (default is 33333333 Hz).

PNP Wrapper function for UART Registration.

This gets called when the chameleon PNP subsystem starts and is called for each Z25/Z125 unit. The Function calls the specific Registration depending if a Z25 or Z125 is found.

Returns:
